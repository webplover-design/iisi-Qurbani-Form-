# Institute of Islamic Sciences, Islamabad — Online Qurbani 2026 / 1447 AH

This is a simple, responsive website for collecting Qurbani orders and skin donation requests.

## Files

- `index.html`: Homepage with hero section and CTAs
- `order.html`: Qurbani order form
- `skin.html`: Skin collection form
- `token.html`: Order confirmation/token page
- `terms.html`: Terms & Conditions
- `privacy.html`: Privacy Policy
- `styles.css`: CSS styles
- `script.js`: JavaScript logic
- `image/`: Directory for images (logo.svg and bg.png added)

## Features

- Responsive design
- Form validation
- Order summary with add/remove items
- Local storage for order data
- WhatsApp integration for sharing order details
- Download/print token

## Setup

Open `index.html` in a web browser to view the website.

- Bank details are dummy and should be updated in `script.js`
- Images are placeholders; replace with actual images
